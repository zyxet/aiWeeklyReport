## 二、重磅推荐

### A类精选：SAGE — 策略感知让运筹优化建模不再"硬编码"

**论文**: arXiv:2605.02545 | **代码**: `github.com/rachhhhing/SAGE` | **License**: Apache-2.0

南京大学与华中科大联合提出 SAGE（StrAteGy-awarE），核心突破是将"策略选择"显式纳入 LLM 的运筹优化建模流程——不是直接生成公式，而是先判断问题类型、选择建模策略，再输出对应约束。

**关键数字**：
- 在 MIPLIB 和 LP Relaxation 两个基准上，pass@1 从 72.7% → 80.3%
- 将运筹学专家的知识结构（问题分类→策略选择→约束生成）编码为 LLM 的推理路径

**开源价值**：官方代码已完整开源，包含策略选择模块和约束生成模块。这是本周 **A类中工程可复现性最高** 的项目，可直接嵌入任何需要数学建模的 Agent 工作流。

### A类精选：HeavySkill — 把"重思考"变成 Agent 的内技能

**论文**: arXiv:2605.02709 | **代码**: `github.com/wjn1996/HeavySkill` | **License**: Apache-2.0

南京大学的 HeavySkill 提出测试时扩展（Test-Time Scaling）的新范式：将复杂推理分解为多个独立推理 Agent 的并行执行，再通过自适应融合选择最优路径。这不是简单的"想久一点"，而是"想得更结构"。

**关键数字**：
- 在六个数学/编码基准上验证，相比单路径 CoT 显著提升
- 核心机制：并行推理 → 多样性收集 → 自适应融合

**开源价值**：HeavySkill 提供的是一个 "skill" 级别的组件——可以插入任何已有 Agent 框架的推理层。对于 Warp 这类终端 Agent 或 TradingAgents 这类多 Agent 系统，HeavySkill 的并行推理可以直接提升复杂决策的质量。

### A类精选：CASA — 给 Agent 装一个"动态权限门卫"

**论文**: arXiv:2605.02682 | **代码**: `github.com/outshift-open/CASA` | **License**: Apache-2.0

蚂蚁集团与新加坡管理大学的 CASA（Continuous Agent Semantic Authorization）针对一个被忽视的问题：当 Agent 可以自主调用工具、访问数据库、执行代码时，传统的 RBAC 权限模型完全失效。CASA 提出"零信任拦截层"，在消息流层面做确定性校验 + 语义匹配，确保 Agent 不会越权。

**关键数字**：
- 五种确定性运行时检查（工具定义完整性、请求授权验证等）
- 语义层：任务提取 → 任务-工具匹配，首次在多轮对话中实现 TBAC

**开源价值**：这是 Cisco Outshift Open 推出的官方实现，包含拦截层代码和扩展后的 ASTRA 数据集。对于任何需要企业级部署的 Agent 系统（尤其是 MCP 服务器场景），CASA 是目前**最完整的安全框架**。

### A类精选：AcademiClaw — 让学生给 Agent 出题

**代码**: `github.com/GAIR-NLP/AcademiClaw` | **License**: Apache-2.0

复旦与港大构建的双语基准，包含 80 个由研究生真实提出的学术任务。与合成基准不同，AcademiClaw 的任务来自真实研究 workflow——读论文、复现结果、对比方法、写综述。

**开源价值**：数据集 + 评估框架已开源。对于教育 Agent、科研助手类产品，这是目前最贴近真实场景的学术能力评估工具。

### A类精选：ORPilot — 自然语言到优化模型的全自动流水线
