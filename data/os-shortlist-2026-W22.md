## 本周主线总结

**主线一：Agent 生产化方法论成型**
- `12-factor-agents` 建立 Agent 从 demo 到生产的架构共识。当行业开始讨论"prompt 与逻辑分离""动作幂等性"时，这标志着 Agent 工程从玩具走向工业级。

**主线二：上下文管理基础设施层全面爆发**
- `OpenViking`（字节跳动，结构化上下文数据库）与 `agentmemory`（独立开发者，记忆持久化）分别从"空间维度"和"时间维度"解决 Agent 的上下文局限。两者互补且同时进入短名单，说明 Agent 基础设施层已进入多维度并行突破期。

**主线三：编码 Agent 竞争从"大厂独占"走向"混战"
- `opencode`（55k Stars，社区驱动）与 Claude Code / Codex CLI / qwen-code 形成四强竞争。独立开发者在该赛道仍有突破空间，市场远未固化。

**主线四：ML 研究 Agent 持续验证**
- `ml-intern`（HuggingFace）和 `cua`（tryCua）分别在"AI 替代研究者"和"Agent 操控电脑"两个方向持续积累工程成熟度，代表 2026 年 Agent 能力边界的两条主航道。
